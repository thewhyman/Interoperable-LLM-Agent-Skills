# Crypto Investor L2 Scoreboard

Research and generate investment-focused scoreboards for Layer 2 (L2) blockchains.

## Overview
This skill autonomously discovers and analyzes top L2 projects in specific sectors (e.g., Privacy, Gaming). It evaluates them based on:
- Adoption/TVL
- Token Value Accrual
- Momentum
- Ecosystem Strength

## Usage
Simply ask your agent:
- "Create a crypto investor scoreboard for Privacy Layer 2s."
- "Analyze top gaming L2s and generate a scored report."

## Output Format
The skill generates a structured Markdown report including:
- Total Sector Score (1-100)
- Leader Breakdown (The Alpha, The Beta, The Wildcard)
- Investment Thesis
- Watchlist Questions
